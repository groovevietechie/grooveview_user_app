GrooveVie POS Download Directory
================================

This directory contains the GrooveVie POS application downloads.

To add the actual POS zip file:
1. Place your groovevie-pos.zip file in this directory
2. The download button on the homepage will automatically serve this file

Current files:
- README.txt (this file)
- groovevie-pos.zip (add your POS application zip file here)
